핵심요구사항) 게시글 목록에서 각 행에 마우스를 올렸을 때(hover) 해당 게시글의 상세 페이지 데이터를 prefetch할 것.

핵심요구사항) Apollo Client의 useQuery를 사용하여 prefetch를 구현할 것.

핵심요구사항) prefetch는 사용자가 실제로 클릭하기 전에 데이터를 미리 가져와서, 클릭 시 즉시 표시되도록 할 것.

구현방법) 
1. boards-list/list/index.tsx 파일에서 Apollo Client의 useQuery를 import
2. 각 게시글 행(tr)에 onMouseEnter 이벤트 핸들러 추가
3. onMouseEnter에서 FETCH_BOARD_DETAIL 쿼리를 prefetch
4. prefetch는 Apollo Client의 client.query() 또는 useLazyQuery를 사용
5. prefetch된 데이터는 Apollo Client 캐시에 저장되어, 실제 상세 페이지 접근 시 즉시 사용됨

주의사항) 
- prefetch는 비동기 작업이므로 에러 처리를 해야 함
- 같은 게시글에 대해 중복 prefetch를 방지하기 위해 ref나 Set을 사용하여 이미 prefetch한 게시글 ID를 추적
- prefetch 실패 시에도 사용자 경험에 영향을 주지 않도록 에러를 조용히 처리
- prefetch는 사용자가 실제로 클릭할 가능성이 높은 항목에만 적용하여 불필요한 네트워크 요청을 방지

기술적 세부사항)
- Apollo Client의 client.query()를 사용하여 prefetch 구현
- client는 useApolloClient() hook을 통해 가져올 수 있음
- FETCH_BOARD_DETAIL 쿼리는 variables에 boardId를 전달해야 함
- prefetch는 fetchPolicy를 'cache-first' 또는 'network-only'로 설정 가능
- hover 시점에 prefetch하되, debounce를 적용하여 너무 빠른 hover 이동 시 불필요한 요청 방지 고려

예시코드구조:
```typescript
import { useApolloClient } from '@apollo/client';
import { FETCH_BOARD_DETAIL } from '@/components/boards-detail/detail/queries';
import { useRef } from 'react';

export default function BoardsList({ boards, onClickDelete, onClickRow, searchKeyword }: BoardsListProps) {
  const client = useApolloClient();
  const prefetchedIds = useRef<Set<string>>(new Set());

  const handleMouseEnter = async (boardId: string) => {
    // 이미 prefetch한 경우 스킵
    if (prefetchedIds.current.has(boardId)) {
      return;
    }

    try {
      // Apollo Client 캐시에 prefetch
      await client.query({
        query: FETCH_BOARD_DETAIL,
        variables: { boardId },
        fetchPolicy: 'cache-first', // 캐시에 있으면 사용, 없으면 네트워크 요청
      });
      
      // prefetch 완료 표시
      prefetchedIds.current.add(boardId);
    } catch (error) {
      // 에러는 조용히 처리 (사용자 경험에 영향 없음)
      console.debug('Prefetch failed for board:', boardId, error);
    }
  };

  return (
    <div className={styles.container}>
      <table>
        <tbody>
          {boards.map((board, index) => (
            <tr 
              key={board._id} 
              onClick={() => onClickRow(board._id)}
              onMouseEnter={() => handleMouseEnter(board._id)}
            >
              {/* ... */}
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  );
}
```

성능 고려사항)
- prefetch는 사용자가 실제로 클릭할 가능성이 높은 항목에만 적용
- 너무 많은 동시 prefetch 요청을 방지하기 위해 debounce 또는 throttle 고려
- 네트워크 상태가 좋지 않은 경우 prefetch를 건너뛰는 로직 고려 (선택사항)
