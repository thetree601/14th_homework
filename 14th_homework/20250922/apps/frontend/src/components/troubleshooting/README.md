# 문제 해결 가이드 (Troubleshooting Guide)

이 폴더는 프로젝트에서 발생한 문제들과 해결 과정을 정리한 문서들을 담고 있습니다.

## 목적

- 문제 발생 시 빠른 원인 파악
- 유사한 문제 재발 방지
- 문제 해결 과정의 학습 및 공유

## 폴더 구조

```
troubleshooting/
├── README.md (이 파일)
├── 001-image-disappears-on-edit.md (이미지 수정 시 사라지는 문제)
└── ... (추가 문제들)
```

## 문제 해결 시 체크리스트

1. **문제 발생 위치 정확히 파악**
   - 어떤 페이지/컴포넌트에서 발생하는가?
   - URL 경로 확인
   - 브라우저 콘솔 로그 확인

2. **관련 파일 찾기**
   - 문제가 발생하는 컴포넌트 파일
   - 관련 hook, mutation, query 파일
   - 상태 관리 파일

3. **원인 분석**
   - 데이터 흐름 추적
   - 상태 동기화 문제 확인
   - API 호출 및 응답 확인

4. **해결 방법 적용**
   - 최소한의 변경으로 해결
   - 다른 기능에 영향 없는지 확인

5. **문서화**
   - 문제 원인 정리
   - 해결 방법 기록
   - 예방 방법 추가

## 문제 해결 시 주의사항

⚠️ **중요**: 문제가 발생하는 정확한 위치를 먼저 확인하세요!

- URL 경로 확인
- 브라우저 콘솔 로그 확인
- 관련 컴포넌트 파일 확인
- 다른 비슷한 컴포넌트와 혼동하지 않기
